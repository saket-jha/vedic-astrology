Kundalee सॉफ्टवेयर में केवल Map1 वाले मोड्यूल, जो संसार का सबसे अच्छा टोपोलोजिकल प्रोग्राम है , के लिए Meridians.mdb डेटाबेस का प्रयोग हुआ है | जो लोग उस मोड्यूल का प्रयोग नहीं करते हैं उन्हें भी कभी-कभी Meridians.mdb के कारण कठिनाई होती है | इस समस्या का समाधान है माइक्रोसॉफ्ट कम्पनी का एक मुफ्त सॉफ्टवेयर इनस्टॉल करना — AccessDatabaseEngine.exe | इसका 32-bit और 64-bit दोनों version माइक्रोसॉफ्ट मुफ्त में बाँटने के लिए देती है | इस लिंक से Access Database engine (32-bit) को डाउनलोड कर लें और फिर AccessDatabaseEngine.kkk को rename करके AccessDatabaseEngine.exe कर दें, उसके बाद run करने पर इनस्टॉल हो जाएगा |
यदि Map1 का उपयोग नहीं करना है तो Meridians.mdb चेतावानी को अनदेखा करके आगे बढ़ जाएँ | उसके बाद भी समस्या हो तो AccessDatabaseEngine.exe इनस्टॉल कर लें |

Compatibility

It will work on Windows-XP, and on all higher versions of Windows, esp. Windows-7 and Windows-10 but only after changing the Compatibility Property of EXE file to Windows-XP-SP2 or Windows-XP-SP3.

After installing two TTF fonts , one must select "Windows XP3" Compatibility by
right-click Phalit.exe > Compatibility > Run this program for Windows XP (Service Pack 3)

In some computers, Meridians.mdb which is Places database created in old version of Microsoft'e Access does not work and creates error message. Ignore this message and go on using this software, but do not use Places database and fill in Longitude and Latitude manually, and remember that the end of Name of Place must contain Time Zone in degree format, for instance ,82E30 for India, and another boxes for Time Zone should also be filled with similar Time Zone in degrees. Updated version of Places database will not depend on Ms-Access and will not face this problem. Some systems will not accept the MDB database of places created in older version of Microsoft-Access. They will need to enter longitude and latitude manually. Later versions will replace this outdated Microsoft-Access file.

WORD Button

After input data entry, clicking 'WORD' button will open Ms-Word and send output to Ms-Word, provided Ms-Word is already installed. Word versions 2003 and 2010 are ok, but Word-2007 prevents programmatic typing into text boxes.
